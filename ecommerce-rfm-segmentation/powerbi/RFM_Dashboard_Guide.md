# 📊 Power BI Dashboard Build Guide
## RFM Customer Segmentation Dashboard

This guide walks you through building a professional 4-panel Power BI dashboard
using the CSV files generated by the Python pipeline.

---

## Prerequisites

- Power BI Desktop (free download: https://powerbi.microsoft.com/desktop/)
- The two CSV files from the `/outputs/` folder:
  - `rfm_scores.csv` — one row per customer
  - `segment_summary.csv` — one row per segment

---

## Step 1: Import the Data

1. Open **Power BI Desktop**
2. Click **Home → Get Data → Text/CSV**
3. Browse to `outputs/rfm_scores.csv` → Click **Load**
4. Repeat for `outputs/segment_summary.csv`

### Verify column types in the Power Query Editor:
| Column | Type |
|---|---|
| CustomerID | Text |
| Recency | Whole Number |
| Frequency | Whole Number |
| Monetary | Decimal Number |
| R_Score, F_Score, M_Score | Whole Number |
| RFM_Total | Whole Number |
| Segment | Text |

---

## Step 2: Create a Relationship

1. Go to **Model View** (icon on left sidebar)
2. Connect `rfm_scores[Segment]` → `segment_summary[Segment]` (Many-to-One)

---

## Step 3: Build Panel 1 — KPI Cards

Add **3 Card visuals** for top-line numbers:

| Card | Measure |
|---|---|
| Total Customers | `DISTINCTCOUNT(rfm_scores[CustomerID])` |
| Total Revenue | `SUM(rfm_scores[Monetary])` |
| Avg Order Value | `AVERAGE(rfm_scores[Monetary])` |

**Formatting tip**: Use Format → Data label → £ currency, bold font, dark background (#1A1D2E).

---

## Step 4: Build Panel 2 — Revenue Donut Chart

1. Insert → **Donut Chart**
2. **Legend**: `segment_summary[Segment]`
3. **Values**: `segment_summary[Total_Revenue]`
4. In **Format → Colors**, manually assign each segment its color:

| Segment | Hex Color |
|---|---|
| Champions | #6C5CE7 |
| Loyal Customers | #00B894 |
| Potential Loyalists | #0984E3 |
| Recent Customers | #74B9FF |
| Promising | #55EFC4 |
| Need Attention | #FDCB6E |
| About To Sleep | #E17055 |
| At Risk | #D63031 |
| Cannot Lose Them | #E84393 |
| Hibernating | #636E72 |
| Lost | #2D3436 |

---

## Step 5: Build Panel 3 — Customer Count Bar Chart

1. Insert → **Clustered Bar Chart**
2. **Y-axis**: `segment_summary[Segment]`
3. **X-axis**: `segment_summary[Customer_Count]`
4. Sort by Customer_Count descending
5. Apply matching segment colors as above
6. Turn off axis titles, add data labels

---

## Step 6: Build Panel 4 — Scatter Plot (Recency vs Monetary)

1. Insert → **Scatter Chart**
2. **X-axis**: `rfm_scores[Recency]`
3. **Y-axis**: `rfm_scores[Monetary]`
4. **Legend**: `rfm_scores[Segment]`
5. **Size**: `rfm_scores[Frequency]`
6. Set Legend colors to match the segment palette

---

## Step 7: Add a Slicer

1. Insert → **Slicer**
2. Field: `rfm_scores[Segment]`
3. Style: Dropdown or Tile
4. This lets viewers filter all panels by segment

---

## Step 8: Design the Dashboard Canvas

**Recommended layout:**
```
┌────────────┬────────────┬────────────┐
│  KPI Card  │  KPI Card  │  KPI Card  │
├────────────┴────────────┼────────────┤
│   Revenue Donut Chart   │ Segment    │
│                         │ Slicer     │
├────────────┬────────────┴────────────┤
│  Bar Chart │  Recency vs Monetary    │
│            │  Scatter Plot           │
└────────────┴─────────────────────────┘
```

**Theming tips:**
- Canvas background: `#0F1117` (dark navy)
- Card background: `#1A1D2E`
- Title font: Segoe UI Bold, white
- Set all visual backgrounds to `#1A1D2E` with 10% transparency

---

## Step 9: Publish & Share

1. Click **File → Publish → Publish to Power BI**
2. Log in with your Microsoft account (free tier works)
3. Copy the share link and add it to your LinkedIn post or GitHub README
